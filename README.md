# 📊 Data Visualization Dashboard

A full-stack web application that transforms raw business data into interactive visual dashboards. Built during my internship at **PRASUNET COMPANY**, this project empowers businesses to make informed decisions with insightful graphs and metrics.

---

## 🚀 Features

- 📁 Upload CSV files to visualize sales, customer, or financial data
- 📊 Dynamic chart rendering using Chart.js, Recharts, and date-fns
- 🔐 JWT-based user authentication (signup/login)
- ☁️ File upload and storage via Cloudinary
- ⏱️ Real-time data updates using 10-second polling
- 🌐 Fully responsive UI for desktop and mobile users

---

## 🛠️ Tech Stack

### 🔷 Frontend

- **React.js** with **Vite** (for fast build and development)
- **React Router DOM** – for navigation
- **Axios** – for API communication
- **Chart.js** (via react-chartjs-2) & **Recharts** – for advanced data visualization
- **chartjs-adapter-date-fns** – for time-based charting
- **React-Bootstrap** & **Bootstrap** – for responsive UI components
- **react-device-detect** – to customize UI based on user devices

### 🔶 Backend

- **Node.js** and **Express.js** – for building REST APIs
- **JWT** – for secure user authentication
- **bcrypt** – for password hashing
- **Multer** & **multer-storage-cloudinary** – for file handling and cloud upload
- **fast-csv** – for parsing CSV data
- **cookie-parser**, **dotenv**, **cors**, **axios**, **date-fns**

### 🗄️ Databases & Storage

- **PostgreSQL** – for storing business data
- **MongoDB Atlas** – for storing user information
- **Cloudinary** – for cloud file storage

---
